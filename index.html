เตรียมพร้อมเชื่อมต่อ Supabase / GitHub Pages

โค้ดนี้บันทึกข้อมูลด้วย localStorage ก่อน เพื่อให้ทดสอบบนเครื่องหรือ GitHub Pages ได้ทันที และมีจุดเตรียมฟังก์ชันสำหรับต่อ Supabase ในอนาคต
ตัวอย่างตาราง Supabase
receiving_records
- id uuid primary key
- company_name text
- surgeon_name text
- surgery_date date
- received_at timestamp
- receiver_name text
- note text
- current_status text
- tools jsonb
- created_at timestamp
การนำขึ้น GitHub Pages
  1. สร้าง Repository ใหม่
  2. อัปโหลดไฟล์นี้ชื่อ index.html
  3. ไปที่ Settings → Pages
  4. เลือก Branch เป็น main และ Folder เป็น /root
  5. เปิด URL ที่ GitHub Pages สร้างให้
